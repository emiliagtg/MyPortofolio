Latest Projects

Creditcard Fraud Detection

This project aims to develop a fraud detection system for credit card transactions using machine learning methods. The system is designed to automatically identify suspicious or unusual transactions, helping financial institutions prevent losses caused by fraudulent activities. By utilizing historical transaction data, the model is trained to recognize patterns commonly found in normal transactions and detect anomalies that may indicate potential fraud. This project utilizes several algorithms, such as Autoencoder (for anomaly detection), Random Forest, and Isolation Forest (for classification and outlier detection). Each model's performance is analyzed to determine the most effective approach for detecting fraud. The project also saves the trained models along with their preprocessing steps, enabling reuse in production systems. Additionally, it includes the option of packaging the system using Docker for efficient and consistent deployment.

Petcare Asst

PetCare Q&A Web App is a web-based application designed to provide automatic answers to common questions about pet care, specifically for cats and dogs. The app allows users to ask questions related to their pets and instantly receive responses based on a pre-defined dataset of question-answer pairs. The system is developed using Python, with the Flask framework handling backend operations such as routing, question matching logic, and data integration. For the user interface, HTML and CSS are used to structure and style the layout, while JavaScript enables dynamic user interactions like sending and receiving messages on the chat page. The HTML templates are rendered using Jinja2, Flask’s built-in templating engine, which allows dynamic content to be embedded into web pages. All Q&A data is stored in a CSV file, which is processed on the server side to match user questions with the most relevant answers.

MediAI Assist

Medisense is an AI-powered web application designed to classify skin diseases based on user-uploaded images. The project aims to provide an early detection tool for various skin conditions by utilizing a deep learning model trained on real-world dermatological datasets. The backend of the application is developed using Python and the Flask framework, which handles server-side logic, image processing, and integration with the trained AI model. The image classification model is built using TensorFlow and Keras, and is saved in .h5 format for easy deployment. Users can upload images of skin conditions, and the system returns a predicted diagnosis based on the AI model’s output.

Image Processing

This project is a digital image processing collection built using the Python programming language. It consists of multiple Python scripts, each designed to perform a specific image processing task such as adjusting brightness and RGB levels, applying bitwise operations (AND, OR, NOT), adding borders, blending two images, and extracting object boundaries. Each function is implemented in a separate file and can be executed individually. The skills applied in this project include Python programming, a solid understanding of digital image processing, and the use of specialized libraries like OpenCV. The main tools used are Python 3.x, the OpenCV library for handling image operations, and a code editor such as Visual Studio Code to write and run the scripts.

President University Enrollment is a digital application designed to simplify the course enrollment process for university students. Users are required to log in or register an account before accessing the main features.
