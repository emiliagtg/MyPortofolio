Latest Projects

Creditcard Fraud Detection

This project is a machine learning + deep learning based fraud detection system, using models such as Autoencoder, Random Forest, Isolation Forest, and XGBoost, and provides a Flask backend interface for demo or manual prediction.
PetCare Web App

The PetCare project is a Flask-based web application that serves as a Question Answering (QA) system to answer questions about pets such as dogs and cats. The system uses an NLP model to convert questions into meaning vectors (embeddings), and then matches them with QA datasets using FAISS (Facebook AI Similarity Search). Users can ask questions naturally, and the system will provide the most relevant answers based on semantic similarity, not just keywords. This project uses technologies such as Python, Flask, Pandas, FAISS, and two transformer models (all-MiniLM-L6-v2 and all-mpnet-base-v2).
Wireless Mobile Programming Enrollment

Project Enrollment Wireless Mobile Programming is a Java-based Android application that provides user login and registration features using Firebase Authentication service. New users can register for an account, while existing users can login to access the main features of the application. All authentication processes are done securely through Firebase. In its development, this project uses Android Studio as the IDE, Java as the programming language, and Firebase as the authentication backend.
Tumor Segmentation

In this project, tumor image segmentation is performed using two different methods: simple thresholding and K-Means clustering. This project aims to explore and compare the effectiveness of both methods in detecting or separating tumor areas in grayscale images.
